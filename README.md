# 导航1024

简洁高效的个人导航网站，支持智能搜索、可视化编辑和丝滑交互。

![导航1024](./image/logo.png)

## ✨ 核心特性

- 🌙 **深色模式** - 一键切换主题，自动保存偏好
- 🔍 **智能搜索** - 站内搜索 + 外部搜索引擎无缝切换
- ☁️ **网盘搜索** - 快捷键 `Ctrl+Enter` 快速搜索
- 🎯 **标签页导航** - 一级/二级分类标签页，丝滑切换动画
- ➕ **可视化编辑** - 无需代码即可管理网址
- 🔄 **拖拽排序** - 卡片、分类自由拖拽排序
- 📱 **响应式设计** - 完美适配手机和电脑
- ⚡ **性能优化** - Logo 秒加载，智能缓存机制

## 🚀 快速开始

### 环境要求
- PHP 7.4+
- Web 服务器（Nginx / Apache / IIS）

### 安装步骤

1. **下载项目**
```bash
git clone https://github.com/yourusername/nav1024.git
```

2. **配置 Web 服务器**
将项目目录设置为 Web 根目录

3. **访问网站**
打开浏览器访问 `http://your-domain.com`

### 默认账号
- **编辑触发密码**: `edit`
- **管理员密码**: `admin123`

> ⚠️ 首次使用后请立即修改密码！

## 📖 使用指南

### 进入编辑模式
1. 在搜索框输入 `edit`
2. 输入管理员密码
3. 点击出现的编辑控制面板

### 智能搜索
- **第一次搜索**: 优先显示站内结果
- **第二次搜索**: 直接使用外部搜索引擎
- **快捷键**: `/` 聚焦搜索框，`Enter` 搜索，`Ctrl+Enter` 网盘搜索

### 标签页导航
- 点击一级分类 → 显示该分类及所有二级分类的标签页
- 点击二级分类 → 焦点落在对应标签，下划线丝滑移动
- 点击"全部" → 显示所有一级和二级分类的内容

### 拖拽排序
1. 进入编辑模式
2. 按住 `⋮⋮` 图标拖动
3. 点击「💾 保存修改」

## ⚙️ 配置说明

编辑 `config.php` 文件：

```php
// 安全设置
define('ADMIN_PASSWORD', 'admin123');      // 管理员密码
define('EDIT_TRIGGER_KEY', 'edit');        // 编辑触发字符串

// 网站设置
define('SITE_TITLE', '导航1024');          // 网站标题
define('BANNER_IMAGE', '');                // Banner 图片（留空使用必应壁纸）
define('SHOW_RECENT_VISITS', true);        // 显示最近访问

// 数据路径
define('DATA_PATH', __DIR__ . '/data/data.json');
```

## 📊 数据格式

数据保存在 `data/data.json`：

```json
[
  {
    "id": 1,
    "title": "网站标题",
    "url": "https://example.com",
    "desc": "网站描述",
    "category": "工具-开发工具"
  }
]
```

**分类命名**: 
- 一级分类: `"工具"`
- 二级分类: `"工具-开发工具"`
